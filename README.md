# Chrono_Frelon
Distribution vers Androïd et IoS d'un chrono quadruple 
L'apiculteur installe iun pot à mèche et marque les frelons qu'il voit avec les crayons marqueyr de reine.
Quatre couleurs sont possible : Rouge, Bleue, Verte et Blanche 
Les quatres chronos sont indépendants et comporte 
- 3 touches : Start/Stop, SUP et RST
- 3 compteurs : les secondes depuis l'appui sur Start/Stop, le nombre d'essais, le temps moyen et la distance moyenne du nid
